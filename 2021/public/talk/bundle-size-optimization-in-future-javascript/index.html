Bundle Size Optimization in Future JavaScript

11/27/2021, 17:30 - 17:40
Track A
Spoken language: ja
Slide language: ja

(Untranslated) Core Web Vitals の登場などにより、フロントエンド開発におけるパフォーマンスチューニングの重要性が増しています。 その中でも、バンドルサイズの削減については Tree-Shaking や Dead Code Elimination が普及していますが、未だ一般的な Class-based OOP 等のプログラミングパターンとは相性が悪く、アプリケーション開発者が設計レベルから意識する必要があります。 この LT では、現在の JavaScript でバンドルサイズが膨らんでしまうパターンを紹介し、それらが将来的に解決される可能性を、TC39 で議論中の仕様やエコシステムの進化などから考察します。
